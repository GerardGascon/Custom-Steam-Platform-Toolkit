# Installation of Custom Platform Toolkit for Steam

> [!NOTE]
> Last Steamworks SDK version tested has been **2025.162.1**

1. Install the required packages
    - com.unity.platformtoolkit
    - com.geri.platformtoolkit.steam
      - You can use git for that, just add this url: https://github.com/GerardGascon/Custom-Steam-Platform-Toolkit.git?path=/Packages/com.geri.platformtoolkit.steam

2. Setup platform toolkit for Steam.
    - Add Steamworks.NET plugin to Unity.
