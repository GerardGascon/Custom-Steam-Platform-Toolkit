# User Platform Capabilities File

From what I've seen, this file can only be edited from a text editor.

> [!NOTE]
> Everything described here is based on code comments found on Platform Toolkit.

| Field                         | Description                                                                                                                   | Values                                                                                                                                                                                                                                                                                                                                                                                                                                                                        |
|-------------------------------|-------------------------------------------------------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| m_MaxSignedInAccounts         | Maximum number of accounts that can be signed in at the same time.                                                            |                                                                                                                                                                                                                                                                                                                                                                                                                                                                               |
| m_PrimaryAccountBehaviour     | Enumerates the different behaviors for the primary account system.                                                            | 0 - The system does not support primary accounts<br>1 - The account must be signed in before the application is launched and cannot be changed afterward<br>2 - Signing in a primary account is not required, but if an account is signed in, it cannot be changed afterward<br>3 - The primary account can be signed in or out at any time and it can be changed                                                                                                             |
| m_AdditionalAccountBehaviour  | Enumerates the different behaviors for additional accounts.                                                                   | 0 - The system does not support additional accounts<br>1 - Additional accounts can be signed in or out at any time<br>2 - Additional accounts can only be signed in after the application has launched using IAccountPickerSystem only, they cannot be signed in from the system environment, however, they can be signed out afterwards (after sign-in) from the system environment. All additional account will be automatically signed out when the application is closed. |
| m_AllowMultipleSignInAttempts | Determines whether the platform allows multiple sign-in attempts. If false, the platform will only allow one sign-in attempt. |                                                                                                                                                                                                                                                                                                                                                                                                                                                                               |
| m_SupportsAchievements        | Determines whether the platform supports an achievement system.                                                               |                                                                                                                                                                                                                                                                                                                                                                                                                                                                               |
| m_AccountsCanManuallySignOut  | Determines whether the platform allows manual sign out of accounts via the SignOut function.                                  |                                                                                                                                                                                                                                                                                                                                                                                                                                                                               |
| m_SupportsAccountInputPairing | Determines whether the platform supports an account to input pairing relationship system.                                     |                                                                                                                                                                                                                                                                                                                                                                                                                                                                               |
| m_SupportsLocalSaving         | Determines whether the platform supports a local saving system not tied to any account.                                       |                                                                                                                                                                                                                                                                                                                                                                                                                                                                               |
