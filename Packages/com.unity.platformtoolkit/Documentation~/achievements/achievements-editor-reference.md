# Achievement Editor settings reference

Use the Achievement Editor to add common achievements to your project.

Navigate to **Window** > **Platform Toolkit** > **Achievement Editor** to open the Achievement Editor.

| **Property** | **Description** |
| :---- | :---- |
| **Id** | Enter a unique ID for the achievement.|
| **Unlock Type** | Select the achievement type to use: <br><ul><li>**Single**: The achievement is unlocked only once.</li><li>**Progressive**: The achievement unlocks at a specific progress point.</li></ul> |
| **Progress Target** | Set the target value to unlock a Progressive achievement type. This property is available only when **Unlock Type** is **Progressive**.  |
| **Import** | Import existing achievement data from a .csv file. For more information, refer to [Import achievement data](import-achievement-data.md). |
| **Export** | Export the current achievement data to a .csv file. For more information, refer to [Import achievement data](import-achievement-data.md). |
