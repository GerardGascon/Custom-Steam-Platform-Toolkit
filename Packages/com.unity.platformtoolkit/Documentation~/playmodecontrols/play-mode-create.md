# Create a Play Mode Controls Settings asset

Use the following steps to create a Play Mode Controls Settings asset:

1. Navigate to **Window** > **Platform Toolkit** > **Play Mode Controls**.
2. Select **Create Play Mode Controls Settings**.
3. Save the asset in your project.
4. Select the behaviors you want to use from the **Platform** dropdown.

> [!NOTE]
> Some platforms have a concept of a system-level user account. Creating a Play Mode Controls Settings asset will also create and configure default test users for use in Play mode.
