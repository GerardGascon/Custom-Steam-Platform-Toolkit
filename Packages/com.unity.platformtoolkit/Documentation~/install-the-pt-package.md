# Install the Platform Toolkit package

To install the Platform Toolkit package, download the package from the [Package Manager](https://docs.unity3d.com/Manual/upm-ui.html) in the Unity Editor.

To [install the package by name](https://docs.unity3d.com/Documentation/Manual/upm-ui-quick.html), use the following string:

* com.unity.platformtoolkit

## System requirements

This version of the Platform Toolkit package is supported by the following versions of the Unity Editor:

* 6.3 and later

## Platform compatibility

For information on supported platforms, refer to [Install platform modules and packages](./install-platform-modules.md).
